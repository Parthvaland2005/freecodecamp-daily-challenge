## 🚀 Getting Started

Follow these instructions to run the challenges locally on your machine.

### Prerequisites

Make sure you have Python installed. You can download it from [python.org](https://www.python.org/downloads/).

### Execution

To run any specific day's challenge, open your terminal and execute the Python file:

```bash
python <filename.py>
```

**Example:**
```bash
python day3_inverted_matrix.py
```
